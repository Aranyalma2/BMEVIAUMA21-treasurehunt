# Vue 3 + TailwindCSS + NestJS + Prisma + PostgreSQL

- **Frontend:** Vue 3 + Tailwind CSS  
- **Backend:** NestJS + Prisma ORM  
- **Database:** PostgreSQL  

---

## 1. Project Overview

This setup follows a monorepo full-stack architecture: (apps/)

```
admin/      → Vue3 + tailwindCSS
frontend/   → Vue3 + tailwindCSS
backend/    → NestJS API + Prisma ORM
database/   → PostgreSQL instance
```

---

## 2. Prerequisites

Make sure you have:

- [Node.js](https://nodejs.org/) 22.16+
- [yarn](https://yarnpkg.com/) 1.22+
